# rust-sniffer

Высокопроизводительный сетевой анализатор пакетов, написанный на языке Rust. Проект предназначен для перехвата сетевого трафика, деагрегации потоков и отображения статистики в режиме реального времени.

## Ключевые возможности

- **Захват трафика:** Реализован с использованием `pnet` для работы с сетевыми интерфейсами.
- **Анализ потоков:** Автоматическая группировка пакетов в потоки (Flow) с учетом направления (A→B и B→A).
- **Агрегированная статистика:** Сбор данных о количестве пакетов, переданных байтах, версиях TLS и DNS-запросах для каждого потока.
- **Модульность:** Разделение логики на захват, парсинг протоколов, логирование и вывод данных.

## Предварительные требования (Windows)

Для корректной работы на Windows требуется установленный **Npcap**.

1. Установите **Npcap** с официального сайта: [https://nmap.org/npcap/](https://nmap.org/npcap/).
2. Скачайте **Npcap SDK** и распакуйте его (например, в `C:\NpcapSDK`).
3. Настройте пути для компилятора Rust. Добавьте в ваш файл `C:\Users\<Ваш_Пользователь>\.cargo\config.toml` следующие строки:

```toml
[target.x86_64-pc-windows-msvc]
rustflags = ["-L", "C:\\NpcapSDK\\Lib\\x64"]
```

Структура проекта

    src/capture.rs — логика перехвата пакетов.

    src/protocols.rs — парсинг сетевых протоколов и структуры данных потоков.

    src/output.rs — формирование и отображение статистики.

    src/iface_names.rs — работа с именами сетевых интерфейсов.

    src/logger.rs — инструменты логирования.
